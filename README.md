## 📋 About

A fully responsive, single-page portfolio built with pure **HTML, CSS & JavaScript** — no frameworks, no dependencies.

---

## ✨ Features

- ⚡ Single HTML file — no build tools needed
- 📱 Fully Responsive (Mobile, Tablet, Desktop)
- 🍔 Hamburger Menu for mobile
- 🎨 Dark modern theme with cyan accents
- 🌀 Smooth scroll animations
- 📬 Contact form powered by **Formspree**
- 🖼️ Profile photo embedded as Base64
